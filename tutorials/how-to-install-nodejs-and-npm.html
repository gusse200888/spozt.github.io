How to install NodeJS and NPM

1. Go to https://nodejs.org and click LTS Recommended For Most Users. It should look something like this:
2. When the download has completed, then open the setup.
3. Click on the checkbox: 'I accept the terms in the License Agreement' and click Next.
4. Choose the path to where you wanna install NodeJS and NPM, then click Next.
5. Click Next and click next again.
6. Click Install and accept the Windows Admin Popup Prompt.
7. Click Finish and Enjoy. You can now use NodeJS and NPM in command prompt by typing (for nodejs): 'node' and for npm: 'npm'
